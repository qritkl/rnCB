# rnCB
Renomeia todos os arquivos encontrados com o CBminer. Pode ajudar em casos de irrastreabilidade.
## como usar
1 - Mova os arquivos que serão renomeados para uma pasta exclusiva com o rnCB.py presente na mesma.
2 - Execute o script e ele criará um diretório com a cópia dos arquivos renomeados.
## como executar
```
python rnCB.py
```
